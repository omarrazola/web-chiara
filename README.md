# Laborarius Landing Page

Esta es la landing page para **Laborarius**, una solución integral para el control horario.

## Estructura del Proyecto

*   `index.html`: Estructura principal de la página.
*   `styles.css`: Estilos visuales y diseño responsive.
*   `assets/`: Carpeta que contiene las imágenes y recursos gráficos.

## Cómo desplegar en GitHub Pages

1.  Sube estos archivos a un repositorio en GitHub.
2.  Ve a la pestaña **Settings** (Configuración) del repositorio.
3.  En el menú lateral, haz clic en **Pages**.
4.  En **Source**, selecciona `Deploy from a branch`.
5.  En **Branch**, selecciona `main` (o `master`) y la carpeta `/ (root)`.
6.  Haz clic en **Save**.

¡Tu web estará lista en unos minutos!
